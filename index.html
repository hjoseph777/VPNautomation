VPN Automation Workflow

1. User Login

Prompt user for login credentials.

2. Connect to CAS-VPN

Execute command to connect to CAS-VPN.

Verify connection is successful.

3. Open Remote Desktop App

Launch the Remote Desktop application.

4. Access Designated Machine

Connect to the designated machine using Remote Desktop.

Verify connection is successful.

5. Work on Machine

User performs necessary tasks on the machine.

6. Disconnect All Connections

Execute command to disconnect from the designated machine.

Execute command to disconnect from CAS-VPN.

7. End Session

Confirm all connections are terminated.

End user session.
